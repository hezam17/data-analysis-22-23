# Retail Data Set:
* Invoice No: Выставление счета

* StockCode : Биржевой код

* Description : Описание

* Quantity : Количество

* InvoiceDate : Дата выставления счета

* UnitPrice : цена за единицу

* CustomerID : Идентификатор клиента

* Country : Страна
